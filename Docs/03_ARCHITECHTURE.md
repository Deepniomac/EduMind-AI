\# System Architecture(25-01-2026)



The project follows a client-server architecture.



User (Browser)

&nbsp;   ↓

React Frontend (UI)

&nbsp;   ↓

FastAPI Backend (Logic)

&nbsp;   ↓

AI APIs (To be integrated)



The frontend is responsible for user interaction and UI rendering.

The backend is responsible for business logic, AI integration, and data processing.
